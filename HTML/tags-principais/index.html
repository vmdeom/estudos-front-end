HTML Básico

  • <!DOCTYPE> Define o tipo de documento
  • <HTML> Define o documento HTML
  • <head> Contém os metadados ee informações do documento
  • <title> Define o título do documento
  • <body> Defineo corpo do documento
  • <h1> a <h6> Representa os cabeçalhos do documento
  • <p> Define um parágrafo
  • <br> Insere uma única quebr de linha
  • <hr> Define uma mudança de tema na página
  • <!--...--> Define um comentário

<!DOCTYPE>

Define o tipo de documento;

Todas os documentos HTML devem começar declarando o <!DOCTYPE>;

A declaração não é uma tag HTML, mas sim uma informação para o Navegador entender do que esse documento se trata.

<HTML>

Define o documento HTML;

A tag <html> representa a raíz do documento;

A tag <html> contém todos os outros elementos HTML(exceto para a tag <!DOCTYPE>).

<head>

Contém os metadados ee informações do documento;

Metadados são dados sobre o documento HTML, esses dados não são exibidos;

os seguintes elementos podem ser incluidos na tag <head>.

  • <title>
  • <style>
  • <base>
  • <link>
  • <meta>
  • <script>
  • <noscript>

<title>

Define o título do documento;

É uma tag requirida em documentos HTML;

O elemento <title>

  • Define o nome da página no navegador
  • Define o títlo para a página quando adicionada aos favoritos
  • Mostra o título da página em mecanismos de pesquisa

<body>

A tag <body> define o corpo do documento;

A tag <body> contém todo o conteúdo de um documento HTML, como heders, paragrafos, imagens, hyperlinks, tabelas, listas, etc.

<h1> a <h6>

<h1> a <h6> representa os cabeçalhos do documento;

Sendo <h1> mais importante e <h6> menos importante.

<p>

<p> define um parágrafo.

<br>

<br> representa uma única quebra de linha;

<br> é uma tag vazia, isso quer dizer que não é necessário fechar essa tag.

<hr>

<hr>Define uma quebra de tema em uma págiana HTML;

O elemento <hr> é exibido como uma linha horizontal.

<!--...-->

É uma tag de comentário no código fonte do documento;

Todo o conteúdo dentro dessa tag não é exibido na página.
